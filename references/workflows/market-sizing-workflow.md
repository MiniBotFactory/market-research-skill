# 市场规模评估工作流

完整的 TAM/SAM/SOM 市场规模评估流程，约需 2-5 个工作日。

---

## 适用场景

- 融资 BP 的市场规模章节
- 新市场进入可行性评估
- 内部业务投资回报论证
- 战略规划的市场预测

---

## 执行流程

### 阶段 0：需求确认（0.5天）

**角色**：需求分析师

关键明确事项：
- [ ] 市场的定义范围（按产品类型/地域/客群）
- [ ] 用途（融资BP/内部决策/公开报告）
- [ ] 精确度要求（数量级/区间/精确数字）
- [ ] 时间维度（当前规模/未来3-5年预测）

**注意**：融资 BP 和内部决策对「市场定义」有不同需求
- 融资 BP：通常需要更大的 TAM，但要能说清楚逻辑
- 内部决策：通常需要更精确的 SAM，有实际参考价值

---

### 阶段 1：市场定义（0.5天）

**角色**：需求分析师 + 首席分析顾问

**必须明确三层市场边界**：

```
TAM 定义：
- 产品类别：[解决什么问题的所有解决方案]
- 地域：[全球/国家/地区]
- 客群：[所有潜在用户]

SAM 定义：
- 产品：[我们实际提供的产品/服务]
- 地域：[我们能实际服务的地区]
- 客群：[符合我们目标画像的用户]

SOM 定义：
- 时间范围：[未来X年]
- 市场份额假设：[基于什么判断]
```

**常见错误**：把 TAM 当成 SAM 声称，夸大市场规模而失去可信度。

---

### 阶段 2：数据收集（1-3天，并行）

**角色**：数据研究员（内部）+ 市场情报员（外部）— 同时启动

**市场情报员重点采集**：

**行业总规模数据**：
- [ ] 行业研究报告（IDC/Gartner/艾瑞/易观等）中的市场规模数字
- [ ] 政府统计年鉴中的相关行业产值
- [ ] 上市公司年报中披露的市场数据

**基础单元数据**（用于自下而上估算）：
- [ ] 目标客户总数（工商数据/行业协会）
- [ ] 行业平均客单价（竞品定价/销售数据）
- [ ] 付费渗透率（行业报告/类比市场）
- [ ] 增速数据（历史CAGR）

**类比市场数据**（如直接数据不足）：
- [ ] 相似市场（美国/欧洲同类市场规模）
- [ ] 相似产品在其他行业的渗透率

---

### 阶段 3：双路径估算（1天）

**角色**：首席分析顾问

参考方法论：`references/methods/market-sizing.md`

**必须完成两种方法**：

**路径 A：自上而下**
```
[行业总规模数据来源]
    × 地域占比（%）= [数字]
    × 行业渗透率（%）= [数字]
    × 目标客群占比（%）= [数字]
SAM 自上而下 = X 亿元
```

**路径 B：自下而上**
```
目标客户数量 = [数字]（来源：[...]）
    × 付费意愿（%）= [数字]
    × 平均客单价 = [数字]
SAM 自下而上 = Y 亿元
```

**交叉验证**：
- 差异 < 20%：取平均，标注区间
- 差异 20-50%：找出原因，说明采用哪个
- 差异 > 50%：重新审视假设和数据

---

### 阶段 4：情景分析（可选）

**角色**：首席分析顾问

对关键假设做敏感性分析：

```
基准情景：使用最可能的假设 → X 亿元
乐观情景：关键假设上调 20% → X+ 亿元
保守情景：关键假设下调 20% → X- 亿元
```

关键假设通常包括：
- 付费渗透率
- 平均客单价增长
- 市场增速

---

### 阶段 5：报告与审核（1天）

**角色**：项目经理 → 质量审核员

使用 `assets/report-templates/market-sizing-report.md`

**市场规模报告必须包含**：
- [ ] TAM/SAM/SOM 的明确定义（不是数字，是边界）
- [ ] 每个估算数字的来源和假设
- [ ] 两种方法的估算过程（透明度）
- [ ] 置信度说明
- [ ] 增速预测的依据

**质量审核特别关注**：
- [ ] TAM 和 SAM 的定义是否合理区分？
- [ ] 所有数据是否标注了来源和年份？
- [ ] 增速假设是否有数据支撑（不能是拍脑袋）？
- [ ] 是否主动说明了估算的局限性？

---

## 关键指标参考

不同类型市场的典型 CAGR（仅供参考，需引用具体数据）：
- 成熟行业：2-5%
- 增长行业：10-20%
- 高速增长（SaaS/AI/新能源）：25-50%
- 衰退行业：负增长

中国各行业渗透率参考（需用具体研究报告替换）：
- 企业SaaS：约 15-25%（较低，增长空间大）
- 电商：约 25-30%（接近饱和）
- 移动支付：约 85%（高度普及）
