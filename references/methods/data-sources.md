## 中国市场数据源

### 政府官方数据（权威，免费）

| 数据源 | 内容 | 网址 |
|--------|------|------|
| 国家统计局 | GDP、人口、行业产值、消费数据 | data.stats.gov.cn |
| 国家企业信用信息公示系统 | 工商注册信息、企业数量 | gsxt.gov.cn |
| 工信部 | ICT行业数据、工业数据 | miit.gov.cn |
| 商务部 | 贸易数据、外资数据 | mofcom.gov.cn |
| 发改委 | 宏观政策、行业规划 | ndrc.gov.cn |
| 证监会/交易所 | A股上市公司年报（免费） | cninfo.com.cn |

**使用技巧**：
- 上市公司年报包含行业数据，是行业分析的重要来源
- 工信部「电子信息产业统计」和「软件业务收入」数据覆盖ICT行业

## 数据质量快速评估

使用前对数据源做快速检查：

```
□ 发布机构：政府/学术/权威行业机构 → 🟢
            知名咨询公司 → 🟡
            不知名机构/个人博客 → 🔴

□ 数据时效：当年或上一年 → 🟢
            2-3年前 → 🟡
            3年以上 → 🔴（仅供背景参考）

□ 方法说明：有样本量/调研方法说明 → 🟢
            无方法说明 → 🟡

□ 覆盖范围：与目标市场定义一致 → 🟢
            定义有差异需要换算 → 🟡
```

## 数据引用格式

**标准引用格式**：
```
[数据内容]（[来源机构]，[报告名称]，[发布年份]）
示例：中国SaaS市场规模为583亿元（艾瑞咨询，《2023年中国SaaS行业研究报告》，2023年Q3）
```

**数据无来源时的处理**：
标注为「[低置信度-估算]」并说明估算方法，不要省略或假装有来源。
