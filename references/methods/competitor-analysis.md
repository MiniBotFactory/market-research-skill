# 竞品分析方法论

本文档提供竞品分析的核心框架和操作指南，供分析师在执行竞品调研时使用。

---

## 框架 1：竞品对比矩阵（最常用）

适用场景：需要系统比较多个竞品的核心维度。

### 标准维度设置

```
竞品对比矩阵模板：

| 维度           | 我方 | 竞品A | 竞品B | 竞品C |
|----------------|------|-------|-------|-------|
| 目标客群       |      |       |       |       |
| 核心产品功能   |      |       |       |       |
| 定价模式       |      |       |       |       |
| 价格区间       |      |       |       |       |
| 主要销售渠道   |      |       |       |       |
| 品牌定位       |      |       |       |       |
| 融资/规模      |      |       |       |       |
| 核心优势       |      |       |       |       |
| 主要劣势       |      |       |       |       |
| 近期重大动作   |      |       |       |       |
```

**评分标准**（可选，用于定量对比）：
- ⭐⭐⭐ 强：行业领先水平
- ⭐⭐ 中：行业平均水平
- ⭐ 弱：低于行业平均

### 维度权重建议

按调研目的选择重点维度：

| 调研目的 | 高权重维度 |
|----------|-----------|
| 制定定价策略 | 定价模式、价格区间、目标客群 |
| 差异化产品 | 核心功能、用户体验、客群痛点 |
| 市场进入评估 | 市场份额、壁垒、渠道 |
| 融资材料准备 | 市场份额、增速、差异化 |

## 框架 2：SWOT 分析

适用场景：评估特定竞品或自身的综合竞争态势。

### 标准 SWOT 模板

```
SWOT 分析：[公司/产品名]

┌─────────────────────┬─────────────────────┐
│     优势（S）        │     劣势（W）        │
│ 内部可控，正面因素   │ 内部可控，负面因素   │
│                     │                     │
│ • [优势1: 具体描述] │ • [劣势1: 具体描述] │
│ • [优势2]           │ • [劣势2]           │
│ • [优势3]           │ • [劣势3]           │
├─────────────────────┼─────────────────────┤
│     机会（O）        │     威胁（T）        │
│ 外部环境，正面因素   │ 外部环境，负面因素   │
│                     │                     │
│ • [机会1: 具体描述] │ • [威胁1: 具体描述] │
│ • [机会2]           │ • [威胁2]           │
│ • [机会3]           │ • [威胁3]           │
└─────────────────────┴─────────────────────┘
```

**延伸分析：SO/WO/ST/WT 策略**

| 战略类型 | 逻辑 | 示例 |
|----------|------|------|
| SO（优势+机会） | 利用优势把握机会 | 用技术优势进入新兴市场 |
| WO（劣势+机会） | 借机会克服劣势 | 通过融资弥补品牌不足 |
| ST（优势+威胁） | 用优势对抗威胁 | 用客户粘性应对新竞争者 |
| WT（劣势+威胁） | 最小化风险 | 避开直接竞争，聚焦细分市场 |

## 框架 3：波特五力分析

适用场景：评估进入某行业/市场的整体竞争烈度，适合市场进入决策。

### 五力评估模板

**力量 1：现有竞争者的竞争强度**
- 竞争者数量：[...]
- 行业增速：[快速增长 → 竞争较小；成熟/衰退 → 竞争激烈]
- 差异化程度：[低差异化 → 价格战激烈]
- 退出壁垒：[高退出壁垒 → 竞争持续激烈]
- 评级：🔴 高 / 🟡 中 / 🟢 低

**力量 2：新进入者威胁**
- 进入壁垒：[规模经济/品牌/监管/资本要求]
- 现有玩家的反应预期：[强烈/温和]
- 评级：🔴 高（容易进入）/ 🟢 低（难以进入）

**力量 3：替代品威胁**
- 主要替代方案：[...]
- 替代品的性价比：[...]
- 用户转换成本：[高/低]
- 评级：🔴 高 / 🟡 中 / 🟢 低

**力量 4：买方议价能力**
- 买方集中度：[集中 → 议价能力强]
- 产品标准化程度：[标准化 → 买方可比价]
- 转换成本：[低 → 买方议价能力强]
- 评级：🔴 高 / 🟡 中 / 🟢 低

**力量 5：供应商议价能力**
- 供应商集中度：[...]
- 替代供应商可得性：[...]
- 供应商的前向整合威胁：[...]
- 评级：🔴 高 / 🟡 中 / 🟢 低

**综合结论**：
```
五力综合评级：[高竞争强度 / 中等 / 低竞争强度]
市场吸引力：[高 / 中 / 低]
关键竞争因素：[2-3个决定胜负的最重要因素]
```

## 框架 4：竞品定位图（感知定位）

适用场景：可视化各竞品在市场上的定位空间，识别市场空白。

### 制作步骤

1. **选择两个最关键的差异化维度**（X轴和Y轴）
   - 示例：价格（低→高）× 功能复杂度（简单→复杂）
   - 示例：B2B导向 ↔ B2C导向 × 专业度（通用→专业）

2. **在坐标系上标注各竞品位置**

3. **识别空白区域**：是机会还是没有需求？

```
高端/复杂
    │
    │     [竞品B]        
    │              [竞品C]
    ├──────────────────────── 高价格
    │  [竞品A]    
    │       
    │          [空白区域?]
低端/简单
         低价格
```

4. **标注我方目标位置**：在图上标出期望的差异化定位

## 分析输出建议

竞品分析报告的核心输出应包含：

1. **竞品概览表**：所有竞品的横向对比（矩阵格式）
2. **TOP 3 关键差异点**：与头部竞品的最重要差距
3. **市场空白识别**：现有竞品未满足的需求或未占据的定位
4. **差异化建议**：基于分析的具体差异化方向
5. **竞争威胁排序**：哪个竞品是最主要的竞争威胁，为什么
